# Space Shooter Game

An arcade-style spaceship shooter game with intense action!

## Requirements

```bash
pip install pygame
```

## Running the Game

```bash
python space_shooter.py
```

## Controls

| Key | Action |
|-----|--------|
| **W** / **Up Arrow** | Move Up |
| **S** / **Down Arrow** | Move Down |
| **A** / **Left Arrow** | Rotate Left |
| **D** / **Right Arrow** | Rotate Right |
| **Space** | Shoot |
| **R** | Restart after game over |
| **ESC** | Quit to main menu |

## Game Features

- **Player Spaceship** - Thrust and maneuver your ship
- **Multiple Enemy Types**:
  - *Basic* - Standard enemies
  - *Fast* - Quick-moving threats
  - *Tank* - Heavily armored, slow enemies
  - *Boss* - Large bosses with multiple health bars
- **Shooting System** - Rapid fire main gun
- **Power-ups**:
  - *Spread Shot* - Triple gun
  - *Rapid Fire* - Faster shooting
  - *Shield* - Additional health
  - *Bomb* - Clear screen
- **Particle Effects** - Explosions when enemies are destroyed
- **Score System** - Points for enemies and combos
- **High Score** - Best score saved locally
- **Difficulty Levels** - Easy, Medium, Hard, Extreme

## Game Objective

Survive as long as possible, defeat bosses, and get the highest score!

Enemies will get faster and spawn more frequently as the game progresses. Use your shield wisely and destroy enemy ships for points.

## Tips

- Stay in cover when possible
- Use power-ups strategically
- Focus on tank enemies for bonus points
- Don't forget to avoid enemy fire

Enjoy the shoot-em-up action!
